# Incident-Management-and-Digital_Forensics_Investigation
Task(s) - content
 
The CEO of ACME Ltd has contracted you, as a forensics analyst, to investigate the company finance director. There are suspicions they have been supplying company confidential information to unapproved 3rd parties. You have been provided with digital evidence that requires forensic analysis:
  
• a Virtual Machine disk image file.
• a Virtual Machine memory capture file.
• a network activity file.  

You are required to investigate the data for evidence of criminality, misconduct or suspicious activity. 

Using any suitable digital forensics techniques, software or processes:
  
1. Analyse the VM disk image to document & report on: 
a. Methods and techniques of investigation. 
b. Basic Machine info, Local account details, etc. 
c. Stored and deleted files. 
d. Recent activity. 
 
2. Analyse the memory capture to document & report on: 
a. Methods and techniques of investigation.  
b. Data recovered.  
c. ‘Live’ activity captured.
 
3. Analyse the network capture to document & report on:
a. Methods and techniques of investigation. 
b. Recent network activity. 
c. Websites visited. 
d. Data Sent/Received. 
 
4. Produce a timeline of events, combined from the pieces of evidence. 

5. Produce a report for the CEO of ACME Ltd with your conclusions on the evidence
for any suspicious activity. This may include (not an exhaustive list): 
a. Accessing data which would be out-with the remit of a Finance Director. 
b. Distributing company confidential data. 
c. Accessing inappropriate content. 
d. Changing/corrupting data. 
e. Risking safety/security of company infrastructure.
